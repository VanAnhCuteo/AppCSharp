1. Tổng Quan Dự Án

🎯

Sứ Mệnh

Khám phá và kết nối tinh hoa ẩm thực địa phương thông qua bản đồ số thông minh, mang lại sự tiện lợi tối đa cho thực khách.

🚀

Mobile First

Ứng dụng di động mượt mà tích hợp WebView hiệu năng cao, tối ưu hóa cho cả Android Emulator và thiết bị thật.

🏗️

Admin Power

Hệ thống quản trị mạnh mẽ giúp chủ quan và quản trị viên kiểm soát dữ liệu, thống kê và cập nhật địa điểm tức thì.

User Profiles

2. Đối Tượng Người Dùng

🍔

Thực khách (User)

Tìm kiếm địa điểm ăn uống, xem đánh giá, nhận hướng dẫn đường đi và ghi lại lịch sử ghé thăm của mình.

🏪

Chủ quán (Owner)

Cập nhật thông tin quán, quản lý thực đơn, theo dõi lượt khách ghé thăm và phản hồi các đánh giá từ người dùng.

👑

Quản trị viên (Admin)

Kiểm duyệt nội dung, quản lý danh mục ẩm thực, gán quyền Role-based và xuất các báo cáo thống kê hệ thống.

Data Structure

5. Cơ Sở Dữ Liệu (MySQL Schema)

Bảng Trường quan trọng Mô tả
categories category_id, category_name Danh mục ẩm thực (Hải sản, Ăn vặt, Đồ nướng).
pois poi_id, category_id, name, latitude, longitude, user_id Dữ liệu chính về các địa điểm.
poi_images image_id, poi_id, image_url Lưu trữ nhiều ảnh cho mỗi địa điểm.
poi_guides guide_id, poi_id, language, description Thuyết minh âm thanh cho từng địa điểm.
users user_id, username, role, email Quản lý tài khoản (Admin, CNH, User).
reviews review_id, poi_id, user_id, rating, comment Đánh giá từ thực khách.
poi_visits visit_id, poi_id, user_id, visit_time Lịch sử ghé thăm địa điểm.
poi_pending_changes change_id, status (pending/approved) Lưu các thay đổi đang chờ phê duyệt.
Evolutionary Design

6. Cơ Chế Decoupling (Tách biệt logic)

Tách biệt Website & API

Hệ thống Admin Dashboard và Mobile API được tách thành hai dự án riêng biệt, giúp việc bảo trì và nâng cấp không gây ảnh hưởng lẫn nhau.

Interface-based Services

Sử dụng Interface trong ASP.NET Core để triển khai Dependency Injection. Dễ dàng thay đổi Logic xử lý dữ liệu mà không cần sửa đổi Controller.

Database Abstraction

Sử dụng Entity Framework Core làm lớp trung gian, cho phép thay đổi hệ cơ sở dữ liệu (từ MySQL sang SQL Server/PostgreSQL) một cách linh hoạt.

Operations

7. Các Phương Thức API

Food Controller

Quản lý truy vấn địa điểm, lọc theo danh mục, xử lý dịch thuật đa ngôn ngữ và thống kê lượt truy cập.

GET /api/Food • GET /api/Food/{id} • POST /visit

Auth Controller

Xử lý quy trình đăng ký, đăng nhập bảo mật và quản lý thông tin profile người dùng toàn hệ thống.

POST /login • POST /register • GET /profile
Safety & speed

8. Bảo Mật & Hiệu Năng

Xác thực JWT

Sử dụng JSON Web Token để bảo mật các endpoint API, đảm bảo chỉ người dùng hợp lệ mới có thể truy cập dữ liệu nhạy cảm.

Bản đồ Vector

Tối ưu hóa Leaflet.js với định dạng Tile Layer hiệu năng cao, giúp việc di chuyển và thu phóng bản đồ luôn mượt mà trên di động.

Data Encryption

Mã hóa mật khẩu người dùng bằng các thuật toán băm (Hashing) mạnh mẽ trước khi lưu trữ vào cơ sở dữ liệu MySQL.

Future Vision

9. Lộ Trình Phát Triển (Roadmap)

🔔

Thông báo (Push)

Tích hợp Firebase Cloud Messaging (FCM) để gửi thông báo ưu đãi từ quán đến thực khách ở gần (Geofencing).

💳

Thanh toán (Payment)

Tích hợp các cổng thanh toán (Momo, VNPay) để đặt món hoặc đặt bàn trực tiếp qua ứng dụng di động.

🤖

AI Recommendation

Sử dụng Machine Learning để gợi ý món ăn dựa trên sở thích và thói quen ăn uống của từng cá nhân người dùng.
